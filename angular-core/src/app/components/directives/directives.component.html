Directives

There are 3 different directives, Components, Attribute and Structural

Component directive
Components are the main type used, they are used within the template to display data, they are the main building block of Angular, in theory everything is a component and angular allows to easily create and resuse compnents
First Directive component These are all components

Attribute directive
Attribute , they are used to change the appearance or behaviour of other elements including another directive. They are used to modify the behaviour of HTML elements, attributes, properties and components. The most common one provided by Angular are: NgClass, NgStyle and NgModel.
NgClass Attribute directive Adds and removes css classes, ideal to use on form validation
